CAT: A toolkit for advanced analysis of chromatin accessibility data

Chapter 2 Getting Started with CAT

2.1 Why use CAT?

There are several ATAC-seq analysis tools available, so why use CAT? Currently these software focus more on QC/pre-processing of data, with little or no downstream analysis involved. Here, our CAT offers features not available in other tools, especially advanced downstream analysis. We also provide extensive data visualization capabilities.

Markmap

Our whole workflow consists of two major parts, one is data preprocessing and QC (cat-snakemake) built on snakemake framework, and the other is downstream analysis based on R package (CAT).

2.2 Getting Set Up

Before you use CAT, use the addAnnotation and addFootprint functions to add the necessary annotations.

library(CAT)
## 
##                       ______             ___           ____________
##                      /      |           /   \         |____    ____|
##                     |  ,----'          /  ^  \             |  |
##                     |  |              /  /_\  \            |  |
##                     |  `----.        /  _____  \           |  |
##                      \______|       /__/     \__\          |__|
## Author: Tao Zhu
## CAT : Version 1.0.0
## If you encounter a bug please report : https://github.com/tzhu-bio/CAT/issues
addAnnotation(gene_bed = "F:/CAT/example/hg19_gene_standard.bed",
              gtf = "F:/CAT/data/bw/hg19.gtf",
              genome_size = "F:/CAT/example/hg19.chrom.size")
addFootprint(corrected_signal = "F:/CAT/example/ATACcorrect/",
             bindetect_result = "F:/CAT/example/BINDetect/")

Where the gene_bed is the gene’s bed file, including chromosome number, start position, end position, and so on. Note that column names are NOT included.
